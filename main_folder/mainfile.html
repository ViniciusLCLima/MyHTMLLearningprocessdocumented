White spaces

If I put 2 or more spaces or break a line in a content, the browser is going to treat it like just one space, like I did just now.

Paragraphs

Paragraphs are made by an opening tag with a "P" letter after its left angle bracket, putting the content of the paragraph you want to be shown on your page, and then putting a closing tag with another "P" Letter after its forward slash.

Bold and Italic

The bold element is made by putting a content between an openning tag and a closing tag, with the first characters of those being a "b", like this: <b>. The Italic element is made the same way, but instead of putting a "b", we put an "i" into it.

The bold element usually contains and emphasizes keywords of a text, while the italic represents words that would be spoken in different manners from surrounding text if it was spoken out loud.

Superscript and subscript

Superscript is made by putting content between a closing tag and an opening tag, with the first characters of those being "sup", subscript is made the same way, but the first characters of its tags are "sub".

The superscript element contains superscript characters(or "caracteres superíndice" in portuguese) and is most commonly used to express suffixes of dates, such as 4th, or for mathematical concepts like raising a number to a power(eg:22), while the subscript element contains subscript characters(or "caracteres sub-índice")

Strong and emphasis

The strong element is used to indicate that a piece of text has great importance in it. Eg: "Advice: don't close the window without saving your changes". It is made by an opening tag, a closing tag, with its first characters being "strong", and a content between them. The Emphasis element is made the same way, but the first characters of the tags are "em", it is used to indicate where the emphasis should be placed in a group of words, like this, and, by default, it will be shown in the browser in italic.

Quotations and the use of the cite attribute in it

There are two elements for quotations, the Blockquote and the <q>.

The <blockquote> element is used for longer quotes, that take up an entire paragraph. It is made of an opening tag, a closing tag, with its first characters being a "blockquote" and a content.

Browsers tend to indent the contents of the <blockquote> element, however you should not use this element just to indent a piece of text — rather you should achieve this effect using CSS.

The <q> element is made the same way, but the first characters of the tags will be just "q". It is used for shorter quotes that takes place within a paragraph.

There's also the possibility of using the cite attribute to indicate the source of the quotation, which was made in the examples above, it will not be shown anywhere on the screen though.

Abreviation and the use of the title atribute with it

The Abbreviation element is made of an opening tag, a closing tag, with its first characters being "abbr", and a content between the tags:<abbr>. It Tells the browser that content is an abbreviation. Eg:HTML

The title attribute can be used with the Abbreviation element to tell the abbreviation's full term. Like: HTML. The full term will be highlighted if we pass the mouse over that abbreviation in the browser.

Ins and Del

The Ins element is made of an opening tag, a closing tag, with its first characters being "ins", and a content between them. It is used to indicate the insertion of a content on the page. It will be shown with an underline in it by the browser.

The Del element is made the same way, with the exception that the first characters of its tags will be "del". It is used to indicate the deletion of a content from the document. It usually will appear with a line on the topside of its letters in the browser

Eg: "Jon is a good bad lawyer."

<s>

The <s> element is made of an opening tag, a closing tag, with its first characters being an "s", and a content between them. It is used to indicate that a content is no longer relevant or is incorrect. It will appear in the browser with a line through its center.

Lists

You can make three types of lists:

  • The unordered list, which is a list that is not ordered, it will be shown in the browser indented, with a bullet point. It is made by:
    1. putting an opening tag with its first characters being "ul"
    2. putting list items between opening and closing tags with their first characters being "li"
    3. putting its closing tag, whose first characters should be "ul" too
  • The ordered list, which is a list that is ordered and that is going to be shown indented and numbered in the browser.
  • The definition List, which is a list of terms and their definitions(there can be more than one definition for each term, and more than one term for a definition too). Like:
    Roses
    A type of flower
    Wednesday
    The 4th day of the week

There can also be a nested list, which is a list inside another list. To do it, just put a list inside an <li> element, as we did above.

Scrambled eggs

Eggs are one of my favorite foods. Here is a recipe for deliciously rich scrambled eggs.

Ingredients

  • 2 eggs
  • 1tbs butter
  • 2tbs butter

Method

  1. Melt butter in a frying pan over a medium heat
  2. Gently mix the eggs and cream in a bowl
  3. Once butter has melted add cream and eggs
  4. Using spatula fold eggs from edge of the pan to the center every 20 seconds (as if you are making an omelet)

Links

Types of links

There are five different types of links, links to other websites, links to other pages inside the same website where the link is, links to a part of a page, links that open a new window and links that open an email program in our system and create a new email with an email address, for us to send.

Links to other websites

To create a link to another website, you have to create an <a> element, which is made of an opening tag, a closing tag, with its first characters being an "a" letter, and a content. To specify where that link is going to get us, we have to put an HREF attribute inside the opening tag of the element, its value should be the absolute URL(which is the address we type in our browser to get into that website) of that site. The <a> element is going to be shown blue and with an underline in the browser.

E.g., Get to google by clicking here!

Links to pages on the same website

To make links to other pages on the same website, the process is the same as creating a link to another website, except that now we're going to use a relative URL to do it.

A relative URL is a short way of telling our browser how to get to the page or file we want from the page we are currently in. If the file we want to get to is inside the same folder of the file we're currently in just put the name of that file. If it's in our folder's child folder, we must put the name of that folder, followed by a forward slash and then, the file name. If it is inside a grandchild folder we just need to put the name of the child folder, a forward slash, the name of the grandchild folder, and then the file name, and so on. Eg:(childfoldername/grandchildfoldername/filename)If we want a URL for files in a parent folder, we must start it with two dats, a forward slash and then the name of the file, if we want a URL for files from our grandparent folder, we must again start it with two dots and a forward slash, but this time we're going two put it another time(eg:../../filename).

To make a link open in a new window, we have to add a target attribute to the opening tag of its <a> element, with its value being "_blank".

E.g., Click to go to google! (opens in a new window)

After that, we just need to create an <a> element with an href attribute, with its value starting with a # followed by the ID attribute value of the element we want that link to send us.

How to add it

To add images, we got to make an image embed element, it's made just of one tag with its first characters being "img", and its last characters are a space and a forward slash. To tell the browser where is the image, we use the attribute src, with its value being a URL. It is also advisable to use an alt attribute to describe with accuracy what is in that image, this is going to be used by screen readers(for people with visual impairments) and search engines. You can also add the title attribute to give more information about it, and the width and height to define its size in pixels, these can be used for the browser to take the image size into account when rendering the page. Like:<img src="URL" alt="image description" width="100" height="100" />
